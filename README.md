# Iot_mini_Project
차량 내부도 미세먼지에서 안전하지 않기 때문에 운전자에게 쾌적한 운전 환경을 제공을 목표로 설정했다.

# 자동차 공조 시스템
차량 내부의 온도, 습도를 실시간으로 측정하여 에어컨을 자동으로 조절합니다. 또한, 미세먼지 농도를 측정하여 RGB 센서를 통해 운전자에게 직관적 정보를 제공해주는 시스템이다.
